# Roadmap de implementación

## Orden de trabajo

1. Bloque 1: base técnica y orquestación local.
2. Bloque 2: auth y roles.
3. Bloque 3: modelo de datos completo.
4. Bloque 4: panel administrativo básico.
5. Bloque 5: MinIO y subida de archivos.
6. Bloque 6-11: identificación, QR, validación y firma.
7. Bloque 12-15: exportaciones, automatizaciones, seguridad y backups.

## Entregable de esta iteración

- estructura inicial del proyecto
- compose con PostgreSQL, MinIO y n8n
- configuración base de frontend y backend
- migración inicial SQL
- documentación y siguientes decisiones
